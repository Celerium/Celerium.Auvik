# Get-AuvikBaseURI

## SYNOPSIS
Shows the Auvik base URI global variable

## SYNTAX

```powershell
Get-AuvikBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-AuvikBaseURI cmdlet shows the Auvik base URI global variable value

## EXAMPLES

### EXAMPLE 1
